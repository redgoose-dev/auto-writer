Auto Text Writer

랜덤으로 글이 쓰여지는 툴입니다.
무언가의 트리거에 의하여 문자가 랜덤으로 돌아가면서 문장이 쓰여지는 애니메이션을 만들 수 있습니다.

English

기본적으로 사용되는 영어문장입니다.

한글

한글로 이루어진 문장입니다.

Color

무작위로 변하는 컬러를 통하여 애니메이션 됩니다.

Example Code

기초적인 사용법에 대한 예제코드입니다.

import autoWriter from 'auto-writer'

autoWriter(document.getElementById('hello'), {
  text: 'Hello Typowriter!'
}).then(function() {
  console.log('done')
})
